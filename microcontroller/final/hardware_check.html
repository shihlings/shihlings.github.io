Hardware Check

Purpose

This program tests hardware connected to the Raspberry Pi Pico. It tests the function of the devices.

Demonstration

The program is initiated by the Raspberry Pi 4B (the touch screen in the upper part of the screen), and the Raspberry Pi Pico runs the program.

The program initiates and does the Button function checks. After the Buttons, it checks the LED Strip, Red LED Light, Yellow LED Light, Green LED Light, PIR Sensor, Ultrasonic Sensor, Potentiometer, and Buzzer. To repeat the test, press any of the buttons at the end of the program.
