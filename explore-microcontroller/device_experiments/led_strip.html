LED Strip

Introduction

The LED Strip used in this test is a WS2812B LED. The LED Strip is a Programmable Input and Output (PIO) device, and it is connected to the Raspberry Pi Pico directly through GPIO/PIO ports. Device specifications can be found here or by clicking the button below.

In this test, the program gradually turns on each color of the LED Light on the LED Strip, then gradually turns off the LED Light on the LED Strip. The LED Strip can be useful for cases that require multiple LED Lights in a row and cases that require brighter LED Lights.

Open Device Datasheet


Demonstration

The program is initiated by the Raspberry Pi 4B (the touch screen in the upper part of the screen), and the Raspberry Pi Pico runs the program.

The program turns on the Blue LED Lights on the LED Strip gradually first and prints "blue" on the Raspberry Pi 4B. Then, it gradually turns off the Blue LED Lights. Afterward, it gradually turns on the Red LED Lights on the LED Strip and prints "red" on the Raspberry Pi 4B. Then, it gradually turns off the Red LED Lights. Next, it gradually turns on the Green LED Lights on the LED Strip and prints "green" on the Raspberry Pi 4B. Then, it gradually turns off the Green LED Lights. After that, the program turns on all of the LED Lights on the LED Strip and prints "white" on the Raspberry Pi 4B. Finally, it gradually turns off all the lights on the LED Strip, and the program exits.
